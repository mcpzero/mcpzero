# mcpzero-cli

The [MCPZERO](https://mcpzero.io) tunnel CLI, distributed via npm. Installing
this package downloads the prebuilt `mcpzero` binary for your platform and puts
the `mcpzero` command on your PATH.

```bash
npm install -g mcpzero-cli
mcpzero --version
```

Equivalent installs: `brew install mcpzero/tap/mcpzero` or
`curl -fsSL https://mcpzero.io/install.sh | sh`.

The library SDKs are published separately as `mcpzero-sdk`.
